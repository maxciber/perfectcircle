
# 🎯 Círculo Perfecto

Juego interactivo donde debes dibujar el círculo más perfecto posible con el dedo o ratón.

## Características
- 🎨 Feedback en tiempo real
- 📊 Puntuación del 0% al 100%
- 🏆 Mejor marca guardada localmente
- 🎚️ 3 niveles de dificultad
- 🎨 Personalización de colores

## Cómo jugar
Dibuja un círculo sin levantar el dedo/ratón y suelta para evaluar. ¡Intenta conseguir el 100%!

## Tecnologías
- HTML5 Canvas
- JavaScript vanilla
- CSS3
